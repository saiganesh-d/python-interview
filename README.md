# Fractal Interview Prep — Python + ReactJS + AWS

**Mode:** 5-day crash · **Level:** Mid (3–6 yrs) · **Stack:** Python/Django · ReactJS · AWS

## How to use this folder

1. **Start with [`STUDY-PLAN.md`](./STUDY-PLAN.md)** — your day-by-day crash schedule. Follow it top to bottom.
2. **Drill the question banks** in [`questions/`](./questions/) — these are your flashcards. Cover the answer, say it out loud, check.
   - [`python.md`](./questions/python.md) — defend your strength
   - [`react.md`](./questions/react.md) — ⚠️ your biggest gap, spend the most time here
   - [`aws.md`](./questions/aws.md) — broaden vocabulary + the deploy architecture
   - [`system-design.md`](./questions/system-design.md) — mid-level expects this
   - [`sql.md`](./questions/sql.md) — shows up in screens
   - [`behavioral.md`](./questions/behavioral.md) — STAR stories, don't wing these
3. **Build & run the project** in [`project/`](./project/) — Django REST + React, with AWS deploy notes. This is your **demo + talk track**. One project you can explain end-to-end beats scattered theory.

## The one thing that wins this interview
> Be able to explain **one full-stack app you built and deployed**, at every layer, including trade-offs.
> Everything in here feeds that goal.

## Daily ritual (every day)
- 30 min: drill flashcards from one question bank (say answers OUT LOUD)
- Track progress by checking boxes in `STUDY-PLAN.md`
- End each day: write down 3 things you were shaky on → review them tomorrow morning
